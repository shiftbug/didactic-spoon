# Outline of `llm.py`

## Imports
- `dotenv` for loading environment variables from a `.env` file.
- `openai` for interacting with the OpenAI API.
- `os` for accessing environment variables from the operating system.

## Environment Configuration
- Loads environment variables using `load_dotenv()`.
- Retrieves the OpenAI API key from the environment variable `OPENAI_API_KEY`.

## OpenAI Client Initialization
- Initializes the OpenAI client with the API key using `OpenAI(api_key=api_key)`.

## Function: `get_completion`
- Ensures that the `task_config` dictionary contains all required parameters.
- Calls the OpenAI API with the parameters unpacked from `task_config`.
- Returns the completion text from the API response.
- Handles exceptions by returning an error message.
