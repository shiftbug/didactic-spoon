# Outline of `src/App.css`

## Root Container Styles
- Sets maximum width, auto margins for centering, padding, and text alignment for the `#root` element.

## User Input Styles
- Defines styles for `.user-input` class with auto height, minimum height, width, minimum width, and centering margins.

## Engine Output Styles
- Sets width, minimum width, and centering margins for `.engine-output` class.

## Prompt Engines Styles
- Applies padding, width, minimum width, and centering margins to `.prompt-engines` class.

## Text Input Styles
- Styles for `.text-input` class including dynamic width calculation, text color, padding, text alignment, display type, font size, margins, and cursor type.

## Engine Header Styles
- Uses Flexbox to style `.engine-header` class with space distribution between child elements and vertical centering.

## Select Dropdown Styles
- Centers `.select` class, sets text color, padding, text alignment, display type, font size, margins, cursor type, and border radius.

## Toggle Engine Button Styles
- Styles for `.toggle-engine` class with flex growth control, margin adjustment, text color, padding, text alignment, display type, font size, margins, cursor type, and border radius.

## Button Styles
- Defines styles for `.button` class including text color, padding, text alignment, display type, font size, margins, cursor type, and border radius.

## Logo Styles
- Sets height, padding, and transition effects for `.logo` class and hover state effects.

## Animation Keyframes
- Defines `@keyframes logo-spin` for rotating elements.

## Media Query for Reduced Motion Preference
- Specifies animation behavior for `.logo` class inside a media query based on the user's preference for reduced motion.

## Card Component Styles
- Applies padding to `.card` class.

## Documentation Link Styles
- Sets text color for `.read-the-docs` class.
